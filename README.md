Getting and Cleaning Data Course Project

The run_analysis.R script performs the Course Project of the Getting and Cleaning Data Course of Coursera.

How it works
------------

Loading of useful libraries in R (reshape2).

Reads the original data.

Read the test and training data files.

Merges the training and the test sets to create one data set.

Extract only the mean and standard devation columns.

Change column names for appropriate labeling.

Creates a new tidy data set selecting only the average value for every measurement.

Export it to "tidy.txt", as required.

Please, read de CodeBook.md and comments in the run_analysis.R script for more detailed information.
