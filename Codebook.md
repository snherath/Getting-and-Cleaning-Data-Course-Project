Code book for Coursera Getting and Cleaning Data course project

The dataset is based in the tidy_data.txt file of this repository.

See the README.md file of this repository for background information on this dataset.

The dataset is explained in the 'Dataset' section below. and the variables of the list is given in parameters section. the transformation of dataset is carried out based in source data which is described in the final section.


Datset

The tidy_data.txt data file is a text file, containing space-separated values. 
The first row contains the variable names of the variables.

The experiments have been carried out with people who performed 6 activities. WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING. Data is collected using a wearable device (Samsung Galaxy S II)

Transformations

The zip file containing the source data is located at https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip.

following steps are followed for the transformation of dataset
1. The training and test sets were merged to create one dataset.
2. The measurements on the mean and standard deviation (i.e. signals containing the strings mean and std) were extracted for each measurement.
3. The activity identifiers (originally coded as integers between 1 and 6) were replaced with descriptive activity names.
4. The variable names were replaced with descriptive variable names
